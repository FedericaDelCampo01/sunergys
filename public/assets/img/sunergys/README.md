# 📁 Guía de Imágenes y Videos - Sunergys

## Estructura de Carpetas

```
sunergys/
├── banner/          # Imágenes del banner principal (home)
├── about/           # Imágenes de la sección "Quienes Somos"
├── services/        # Imágenes de servicios
├── portfolio/       # Imágenes de proyectos/instalaciones
├── team/            # Fotos del equipo
└── videos/           # Videos (formato .mp4 o links de YouTube)
```

## 📸 Imágenes Necesarias

### 1. Banner Principal (`banner/`)
**Cantidad:** 2-3 imágenes
**Tamaño recomendado:** 1920x1080px o similar (landscape)
**Formato:** JPG o PNG
**Nombres sugeridos:**
- `banner-1.jpg` - Imagen principal con paneles solares
- `banner-2.jpg` - Segunda opción del carrusel
- `banner-3.jpg` (opcional) - Tercera opción

**Descripción:** Imágenes de instalaciones solares, paneles en techos, o paisajes con energía solar.

---

### 2. Sección "Quienes Somos" (`about/`)
**Cantidad:** 2 imágenes
**Tamaño recomendado:** 800x600px o similar
**Formato:** JPG
**Nombres sugeridos:**
- `about-1.jpg` - Imagen principal (lado izquierdo)
- `about-2.jpg` - Imagen secundaria (superpuesta)

**Descripción:** Fotos del equipo trabajando, instalaciones, o instalaciones solares.

---

### 3. Servicios (`services/`)
**Cantidad:** 3 imágenes (una por servicio)
**Tamaño recomendado:** 600x400px
**Formato:** JPG
**Nombres sugeridos:**
- `residencial.jpg` - Instalación residencial
- `empresarial.jpg` - Instalación empresarial/industrial
- `rural.jpg` - Proyecto rural/agrícola

**Descripción:** Imágenes representativas de cada tipo de servicio.

---

### 4. Portfolio/Proyectos (`portfolio/`)
**Cantidad:** 6-9 imágenes
**Tamaño recomendado:** 800x600px
**Formato:** JPG
**Nombres sugeridos:**
- `proyecto-1.jpg` - Instalación residencial
- `proyecto-2.jpg` - Proyecto agrícola
- `proyecto-3.jpg` - Instalación industrial
- `proyecto-4.jpg` - Energía solar
- `proyecto-5.jpg` - Mantenimiento
- `proyecto-6.jpg` - Soluciones rurales
- (más si tienes)

**Descripción:** Fotos de proyectos reales completados.

---

### 5. Equipo (`team/`)
**Cantidad:** 3-4 imágenes
**Tamaño recomendado:** 400x400px (cuadradas)
**Formato:** JPG o PNG
**Nombres sugeridos:**
- `bautista-de-castro.jpg` - Gerente General
- `margarita-navarro.jpg` - Asesora Legal
- `liesvy-delgado.jpg` - Ingeniero Eléctrico
- (más miembros si hay)

**Descripción:** Fotos profesionales del equipo (formato cuadrado, fondo neutro preferible).

---

## 🎥 Videos

### Carpeta `videos/`
**Opciones:**
1. **Subir archivos MP4** (máx. 50MB recomendado)
   - `video-about.mp4` - Video de la empresa
   - `video-instalacion.mp4` - Proceso de instalación

2. **O usar links de YouTube/Vimeo** (más recomendado)
   - Solo necesitas el ID del video de YouTube
   - Ejemplo: Si el link es `https://www.youtube.com/watch?v=ABC123`, el ID es `ABC123`

---

## ✅ Checklist

- [ ] 2-3 imágenes para banner
- [ ] 2 imágenes para "Quienes Somos"
- [ ] 3 imágenes para servicios
- [ ] 6+ imágenes para portfolio
- [ ] 3-4 fotos del equipo
- [ ] Video (opcional) - YouTube ID o archivo MP4

---

## 📝 Notas

- **Optimización:** Comprime las imágenes antes de subirlas (usa herramientas como TinyPNG o ImageOptim)
- **Nombres:** Usa nombres descriptivos sin espacios ni caracteres especiales
- **Formato:** JPG para fotos, PNG para logos/iconos con transparencia
- **Tamaños:** Las imágenes se redimensionarán automáticamente, pero tamaños grandes funcionan mejor
